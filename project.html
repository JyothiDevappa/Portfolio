TuneHub (Music Streaming Application)
TuneHub is a music streaming web application built using Java, Spring Boot, Thymeleaf, HTML, and CSS. It allows users to register as admins or customers. Admins can add and manage songs and playlists, while customers can browse music, create their own playlists, and enjoy seamless music playback.
